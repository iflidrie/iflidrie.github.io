基于PicGo搭建的GitHub图床

发表于2023-08-20|更新于2023-09-09
|阅读量:
1. 首先你得有一个GitHub账号。注册GitHub就不用我多言了。
2. 新建一个仓库

记下你取的仓库名。

3. 生成一个token用于PicGo操作你的仓库：

访问：https://github.com/settings/tokens

然后点击Generate new token。

注意：这个token生成后只会显示一次！你要把这个token复制一下存到其他地方以备以后要用。

4. 配置PicGo

注意：仓库名的格式是用户名/仓库，比如创建了一个叫做img的仓库，在PicGo里我要设定的仓库名就是username/img。一般我们选择master分支即可。然后记得点击确定以生效，然后可以点击设为默认图床来确保上传的图床是GitHub。

更多图床设计：请参考PicGo官方手册
